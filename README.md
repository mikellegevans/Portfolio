# Mikelle Evans — Engineering Portfolio (multi-page site)

## Pages
- index.html (Home — hero with headshot + background image)
- projects.html (project list) + projects/*.html (one page per project)
- skills.html, education.html, about.html

## Add your photos (images/ folder)
Header:
- `headshot.jpg`  — your headshot (square works best, shown in a circle)
- `hero-bg.jpg`   — wide background image behind the header (a shop/project photo is great)

Projects (each page uses two):
- `greenhouse-1.jpg`, `greenhouse-2.jpg`
- `harness-1.jpg`, `harness-2.jpg`
- `airengine-1.jpg`, `airengine-2.jpg`
- `pinball-1.jpg`, `pinball-2.jpg`

Until a photo exists, a clean "Add Photo" placeholder shows instead. No broken images.

## Preview
Double-click index.html to open in your browser. The fade-in animations and hover effects only work in a browser (not in a plain file preview).

## Publish free on GitHub Pages
See PUBLISH.md.

## Customize
- Text: edit any .html file in a text editor.
- Colors: edit the :root variables at the top of style.css.
- The accent color is --accent. The header darkness is --hero-overlay.
