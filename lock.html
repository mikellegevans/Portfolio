Digital Combination Lock

Digital & Micro Systems · RIT
2025

A combination lock built entirely from digital logic on an FPGA, enter the right code in binary and the display reads pass, get it wrong and it reads fail.

What I did

For this two-person project, I designed a digital combination lock on an Altera FPGA board using Quartus. I built the logic with schematics, wiring up flip-flops and logic gates so the circuit could take in a code, hold it, and check it. The board's switches are the input: a switch down is a 0 and a switch up is a 1, so you enter the code in binary. Flipping switch 9 locks in the code, and pressing the clock button registers it. The seven-segment display then shows whether the code passed or failed.

How I tested it

Before running it on the board, I simulated the design with waveforms in Quartus to confirm the logic behaved the way it should, that the right code produced a pass and anything else produced a fail. Then I assigned the pin inputs and outputs to map my logic onto the board's physical switches and display, and tested it live by entering different codes and watching the display respond.

What I took from it

This project made digital logic real for me. Designing a working lock meant thinking through sequential logic and flip-flops, mapping a design onto real hardware, and verifying it in simulation before trusting it on the board. It is also the project that made Digital & Micro Systems my favorite class.
